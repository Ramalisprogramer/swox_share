# Swox Share

Fast, open-source local file sharing app for Android using Wi-Fi Hotspot and TCP.

## ✨ Features

* ⚡ Fast local file sharing
* 📶 Works without an internet connection
* 📷 QR code connection
* 🌍 Multi-language support
* 📜 Transfer history
* ❤️ Ko-Fi support

## 📱 Permissions

Swox Share requires:

* Nearby Devices
* Precise Location

These permissions are only used for discovering devices and transferring files over the local network.

## 🚀 Build

```bash
git clone https://github.com/Ramalisprogramer/swox_share.git
cd swox_share
./gradlew assembleDebug
```

Or open the project in Android Studio and build it normally.
